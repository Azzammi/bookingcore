    Thank you for used our product Booking Core

    Version 1.3.0 - Aug 30 - 2019

     

    1. Availability calendar for Tour

    Navigate to Dashboard > Tour > Availability: Add price by type of Person

    Remember the Max Guest: if you leave it blank, Your client can't book on that day

    2. Tour Detail

    + Live Chat for Customer and Vendor

    + Show Total Price in Tour's Booking form

    + Wishlist for User Panel

     

    3. Tour & Space Settings

    Add Booking fees for space and tour, such as: tax fees, Insurance fees, booking fees.. All this fees will come to Admin Amount

     

    + Smart search location for space and tour: Just need to input 2-3 characters, A list of suggested locations will be displayed, if you don't like this function, you can select Normal Search

    + Allow admin to duplicate content for tour and space: create content easier

     

    4. Vendor System

    + Commission system for Vendor: Navigate to Settings > Vendor Settings

     

    + Page Register for vendor

    + Vendor Profile

     

    5. Location Detail Page

    + Update filter get featured item in Block Template

     

    - Fix:

    + [Fixed] Can't embed youtube iframe in the editor

    + [Fixed] Translate tool shows wrong count number(%)

    + [Fixed] Don't show next button on the calendar

    + [Fixed] Multi-lang bug in Template builder
